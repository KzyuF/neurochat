# neurochat

Telegram-бот для автоматической генерации комментариев к постам в каналах с помощью Groq LLM (Llama 3.3 70B).

## Установка

```bash
git clone https://github.com/KzyuF/neurochat.git
cd neurochat
pip install -r requirements.txt
```

## Настройка

1. Задайте переменную окружения с ключом Groq API:
   ```bash
   export GROQ_API_KEY="your_groq_api_key_here"
   ```
2. Откройте `channels.txt` и добавьте каналы для мониторинга (по одному `@username` на строку).
3. При необходимости измените параметры в `config.py` (задержки, лимит комментариев, промпт).

## Запуск

```bash
python main.py
```

При первом запуске потребуется авторизация:
- Введите номер телефона
- Введите код подтверждения из Telegram
- При наличии двухфакторной аутентификации — введите пароль

Сессия сохраняется в файле `neurochat_session.session`, повторная авторизация не требуется.

## Как работает

- Мониторит новые посты в указанных каналах
- Генерирует комментарий через Groq API (Llama 3.3 70B)
- Ждёт случайную задержку (5–15 минут) перед отправкой
- Публикует комментарий к посту
- Лимит: не более 25 комментариев в день (сбрасывается в полночь)

## Конфигурация

| Параметр | Описание | По умолчанию |
|---|---|---|
| `MIN_DELAY` | Минимальная задержка (сек) | 300 |
| `MAX_DELAY` | Максимальная задержка (сек) | 900 |
| `MAX_COMMENTS_PER_DAY` | Лимит комментариев в день | 25 |
| `GROQ_MODEL` | Модель Groq | llama-3.3-70b-versatile |
